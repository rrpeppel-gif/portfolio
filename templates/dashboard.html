Cyber Security Project

Security Dashboard

Een web-based security dashboard gebouwd met Flask, MySQL en Python. Het systeem bevat gebruikersbeheer, role-based authentication, data visualisatie en security monitoring van netwerklogs.

Over het Project

Dit project is ontwikkeld als een security monitoring platform waarin verschillende gebruikers toegang krijgen tot specifieke dashboards.

Het systeem analyseert logdata uit een database en visualiseert netwerkverkeer, HTTP-statuscodes en verdachte activiteit via grafieken.

Daarnaast bevat het platform een secret management systeem, gebruikersbeheer en role-based access control.

Belangrijkste Functionaliteiten

Authenticatie & Rollen

Gebruikers loggen veilig in via bcrypt password hashing en worden automatisch doorgestuurd naar hun eigen dashboard.

  • Flask Sessions
  • bcrypt hashing
  • Role-based access control
  • User / Admin / Security Manager

Security Monitoring

Analyse van HTTP logs en netwerkverkeer via dynamische grafieken.

  • HTTP statuscodes
  • Top IP-adressen
  • Verkeer per dag
  • HTTP methodes

ETL Pipeline

CSV-logbestanden worden automatisch verwerkt en opgeslagen in MySQL.

  • Data validatie
  • Batch processing
  • Pandas data analyse
  • MySQL integratie

Technologieën

Python Flask MySQL Pandas Matplotlib HTML CSS Cyber Security Data Analyse ETL bcrypt Jinja2

Technische Highlights

Flask Blueprints voor modulaire architectuur
Role-based authentication met sessiebeheer
Data visualisatie met Pandas & Matplotlib
ETL verwerking van logbestanden naar MySQL database
CRUD functionaliteit voor gebruikersbeheer

Wat ik heb geleerd

Tijdens dit project heb ik ervaring opgedaan met backend development, database ontwerp, security principes en data visualisatie.

Daarnaast heb ik geleerd hoe Flask Blueprints gebruikt worden voor schaalbare applicaties en hoe logdata geanalyseerd kan worden binnen een security omgeving.
